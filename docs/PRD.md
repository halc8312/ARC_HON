# PRD: Proximity VC EN->JA Subtitle Overlay (MVP)

## Goal
Windows上でゲームの出力音声から付近VC等の英語音声を検出し、
英語(EN)と日本語(JA)の字幕を画面任意位置にオーバーレイ表示する。

## Non-Goals (MVP)
- ゲームプロセスへの注入/フック/DirectXオーバーレイはしない
- 話者分離（誰が喋ったかの識別）はしない
- 音声保存・録音ファイル保存はしない（デバッグを除きオフがデフォルト）

## Constraints
- 完全ローカルで完結（無料）。クラウドAPIは使用しない
- アンチチート配慮: OSループバック + 外部ウィンドウ表示のみ
- 入力: PCの出力音声（ヘッドホン）

## MVP User Experience
- 喋り終わり（無音検出）後、1〜2秒程度で字幕が出ることを目標
- 表示: 上段EN / 下段JA
- 位置: ドラッグで移動、保存
- 表示: 数秒保持してフェードアウト
- 誤爆対策: 翻訳処理の一時停止トグル（ホットキー）

## Metrics (MVP acceptance)
- 通常会話で実用的に読める（多少の誤訳/遅延は許容）
- 戦闘音で誤爆する場合があっても、停止トグルで運用可能
